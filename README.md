## About

I am a backend engineer at AppsCode working on KubeDB, a Kubernetes-native database management platform. My work focuses on building production-grade operators and automation for running databases reliably on Kubernetes.

Recently I have been working on SAP HANA/HanaDB support in KubeDB, including operator reconciliation, CRDs, webhooks, high availability, RAFT-based coordination, automated system replication, failover handling, monitoring, TLS, and day-2 operations.

## Current Work

- Building Kubernetes operators using Go, controller-runtime, and Kubernetes APIs
- Developing SAP HANA/HanaDB lifecycle management for KubeDB
- Implementing high availability with RAFT-based leader election and coordinator workflows
- Automating SAP HANA system replication, primary takeover, failover, and secondary re-registration
- Adding operational workflows such as scaling, volume expansion, reconfiguration, TLS, and credential rotation
- Integrating Prometheus, ServiceMonitor, health checks, and Grafana dashboards for database observability
