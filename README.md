# Zobayer Abedin

Backend Engineer focused on Kubernetes operators, Go, and cloud-native database automation.

I work at AppsCode on KubeDB, building operator-driven workflows for running production databases on Kubernetes. My recent work focuses on SAP HANA/HanaDB: provisioning, high availability, RAFT-based coordination, system replication, failover, observability, TLS, and day-2 database operations.

## What I Work On

- Kubernetes operators and controller-runtime reconciliation
- SAP HANA/HanaDB automation for KubeDB
- High availability, failover, leader election, and system replication
- CRDs, webhooks, generated clients, Helm charts, and catalog packaging
- Prometheus/Grafana observability for database workloads

## Recent Impact

- Built HanaDB operator components for KubeDB using Go and Kubernetes APIs
- Implemented RAFT-based leader election and coordinator workflows for HanaDB clustering
- Added automated SAP HANA system replication, primary takeover, failover handling, and secondary re-registration
- Developed day-2 operations including vertical scaling, volume expansion, restart, reconfiguration, TLS/reconfigure-TLS, certificate handling, and credential rotation
- Integrated Prometheus exporter support, ServiceMonitor resources, health checks, and system replication status evaluation
- Contributed HanaDB CRDs, webhooks, catalog versions, shared client support, samples, and installer chart updates
